# Shady's Grill - Site Vitrine JO Dakar 2026

## 📌 Description
Ce projet est un site vitrine simple et responsive pour **Shady's Grill**, situé sur la Route de l’Aéroport (en face CBAO, 50 m après la station Shell).  
Il est conçu pour accueillir les visiteurs pendant les Jeux Olympiques de la Jeunesse Dakar 2026.

## 🎯 Objectif
Offrir une vitrine professionnelle et bilingue (Français ↔ Anglais) aux visiteurs internationaux et locaux, afin de booster la visibilité et les réservations pendant l’événement.

## ⚙️ Fonctionnalités
- Page unique (HTML/CSS) avec sections : Accueil, Menu, Localisation, Contact, Réservation
- **Formulaire de réservation en ligne**
- **Bouton WhatsApp direct** pour réserver en un clic
- **Traduction automatique FR ↔ EN** via Google Translate
- Intégration Google Maps pour guider les clients
- Design adapté aux mobiles
- Paiement local (Wave, Orange Money)

---

## 🚀 Étapes pour personnaliser
1. **Nom & slogan** → Modifier dans `index.html` (balise `<header>`).
2. **Images** → Remplacer les fichiers dans `assets/images/`.
3. **Contacts** → Mettre à jour téléphone, email et lien WhatsApp dans `index.html`.
4. **Carte Google Maps** → Remplacer l’iframe par le lien exact de Shady’s Grill (depuis Google Maps → Partager → Intégrer une carte).
5. **Couleurs & styles** → Adapter dans `css/style.css`.
